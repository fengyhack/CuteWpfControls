# CuteWpfControls #

**In brief**

This control library is used for WPF .NET 4.0+

Examples can be found at [WpfControlsDemo](https://github.com/fengyhack/WpfControlsDemo)

**How to install**

```
PM> Install-Package CuteWpfControls
```

**What is in it**

- Controls
  - **LoadingIndicator** (*more styles*)
  - BusyMask
  - ImageButton
  - Timeline
  - Flyout
  - TextBoxEx
  - IPAddressBox
  - IconTextBox
  - **TitleTextBox** (*new feature*)
  - PasswordBoxEx
  - **Pie**
  - **Gauge**
  - **StepBar**
  - **Badge** (*new feature*)
- GenericStyles
  - FlatButton
  - FlatCheckBox
  - FlatProgressBar
  - **FlatScrollViewer**
